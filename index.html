What is MinecraftSL?

MinecraftSL is a 3rd Party Minecraft Launcher with many 3rd party tools inside like an debug center, loading mods per launcher and level backup utility.

Expect changes. Verry soon.

We still look for collaborators on the open source project. Contact "us" via one of the discussion issues on the GitHub source!
